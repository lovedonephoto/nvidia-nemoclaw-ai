NemoClaw Technical FAQ

How do I install NVIDIA Nemo Claw?

Download the professional setup kit from our secure links and run the onboard.sh script. It automatically detects your NVIDIA GPU, installs relevant CUDA drivers (12.x+), and configures the environment for the Nemotron 3 model family.

Can I run Nemotron 3 Super locally?

Yes, provided you have at least 80GB of VRAM (H100) or use a 4-bit quantized version via llama.cpp or vLLM on consumer RTX 4090/5090 hardware.
